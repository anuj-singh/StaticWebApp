About Weather App

Weather App is a simple web application that provides real-time weather updates for major Indian cities. Using the OpenWeatherMap API, our app displays current temperature, weather conditions, and more in a user-friendly format.

  • Live weather data for Indian cities
  • Modern, responsive design using Bootstrap
  • Easy navigation and quick access to weather information
  • Powered by JavaScript and OpenWeatherMap API

Our mission is to make weather information easily accessible and visually appealing for everyone.
